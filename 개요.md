# 장비 시뮬레이터
PLC, 저울 등 장비의 상태를 시뮬레이션하여 시뮬레이션 환경에서 제어를 할 수 있도록 하는 시뮬레이터를 구현한다.

## 개발언어
- Python    

## 시뮬레이션 대상 장비
- PLC
- 저울
-

# 구현 해야될 기능
- 한개의 장비에 1개 이상의 데이터 포맷이 있을 경우 사용자가 선택하게 한다.
- 저울의 경우 ZERO, TARE 명령어에 대한 처리를 한다.
- 저울의 경우 Print Mode (Stream 방식, Command 방식) 를 사용자가 선택하게 한다.
  Stream 방식은 중량값을 계속 전송
  Command 방식은 클라이언트가 중량 데이터 요구 Command 를 보내 왔을 때만 중량값 전송
- 클라이언트로 부터 들어오는 COMMAND를 화면에 표시한다.
- 클라이언트에게 전송되는 RESPONSE를 화면에 표시한다.
